### 5.1 因子

```bash
# 默认含 mom/vol/turn/ep/bp
.venv/bin/python -m qdata.jobs.compute_factors \
  --start 2026-07-01 --end 2026-07-15 --version v1
# 仅价值因子：
.venv/bin/python -m qdata.jobs.compute_factors \
  --start 2026-07-01 --end 2026-07-15 --factors ep,bp --version v1
```

约定：`ep`/`bp` = PIT `net_profit`/`equity` ÷ (`total_mv`×10000)；利润取最新已公告报告期（非强制 TTM）。

### 5.2 回测读数

```python
from datetime import date
from qdata.api.data_api import DataAPI
from qdata.research.portfolio import RebalanceSpec, build_weight_series

api = DataAPI(version="2026-07-15")
w = build_weight_series(
    date(2026, 7, 1), date(2026, 7, 15),
    api=api,
    spec=RebalanceSpec(universe="000905.SH", factor="mom_20", top_n=50),
)
print(w.attrs["dataset_version"], w.groupby("trade_date")["weight"].sum())
```
