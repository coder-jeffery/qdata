# qdata 量化交易终端 — 高保真 UI 设计规范

> 更新日期：2026-07-19  
> 原型：[`ui/trading-terminal/index.html`](../ui/trading-terminal/index.html)  
> 定位：PC 端机构级量化交易 / 研究终端（Figma 级高保真可交互原型）

---

## 1. 产品定位与风格

| 项 | 说明 |
|----|------|
| 场景 | 专业私募 / 机构量化：行情、策略、下单、回测、风控、流水 |
| 主题 | **深色为主**（蓝灰 `#0A1018`）、浅色冷灰日间为辅 |
| 风格 | 冷色投行风：钢青点缀、图表主导、冷静克制、低噪 |
| 质感 | 细线分隔、弱高光、轻阴影；主按钮哑光钢青；无霓虹/厚浮雕 |

---

## 2. 设计规范（对齐 Figma）

| 规范 | 取值 |
|------|------|
| 栅格 | **8px** 基础尺寸系统 |
| 圆角 | 统一 **10px**（小组件内 6–8px） |
| 分割线 | **2px** |
| 阴影 | 轻量化弥散软阴影分层（静止 / 悬浮） |
| 面板 | 半透明 + `backdrop-filter: blur(14–16px)` |
| 控件 | 哑光金属渐变底 + 弱高光内描边 |

### 2.1 色彩（低饱和）

**深色（默认）**

| 角色 | 色值倾向 |
|------|----------|
| 底色 | 蓝灰 `#0A1018` / 侧栏 `#0D1520` / 面板 `#121C2A` |
| 点缀 | 钢青 `#8EC4D4` / 高光 `#B0DBE8`（激活、主按钮、曲线） |
| 盈利 / 亏损 | `#4ED68A` / `#F07178`（高可读） |
| 文字 | `#F4F8FC` / `#A8B4C4` / `#6A788A` 三级 |

**浅色日间**：底 `#F0F2F4`，面板白半透明，主色同系压深，盈亏同色相降饱和。

### 2.2 字体与图标

| 层级 | 用法 |
|------|------|
| 标题 | 无衬线 SemiBold / Bold（IBM Plex Sans + Noto Sans SC） |
| 行情数字 | JetBrains Mono · Medium · tabular-nums；重点价弱高光 |
| 辅助注释 | 浅灰 11–12px |

图标：单色细线金融图标（1.5px stroke）；图表为极简折线 / 柱状示意。

| 模块 | 功能 |
|------|------|
| 左侧导航 | 交易 / 研究 / 策略库 / 资金流水 / 风险；日志显隐 |
| 顶部工具栏 | 账户切换、时间周期、骨架/空态演示、设置下拉、通知、主题 |
| 行情自选栏 | 多标的价与涨跌；选中联动图表与下单代码 |
| 多窗分屏 | 2×2 主工作区；策略库/流水为双栏视图 |
| K 线面板 | 分时 / 日线 / 60 分 / 指标 Tab；OHLCV 中文标签 |
| 策略编辑 | 代码区 + TopN/窗口/换手滑块 + 模板 Chip |
| 持仓统计 | 总资产 / 市值 / 现金 / 当日盈亏 |
| 委托下单 | 买卖切换、价量、限价；提交前风险检查 |
| 历史成交 | 时间 / 方向 / 价量 / 费用 |
| 回测结果 | 夏普、回撤、年化、净值折线、区间表 |
| 风险弹窗 | 遮罩 + 弱红提醒列表 |
| 资金流水 | 入金 / 买卖 / 结息 |
| 策略模板库 | 动量 / 价值 / 行业中性等载入 |
| 日志侧栏 | INFO / FILL / REJECT / RISK |

## 4. 交互状态（原型已覆盖）

| 状态 | 表现 |
|------|------|
| Hover | 导航行、按钮、自选条高亮；边框偏主色 |
| Active | 导航 / Tab / 买卖方向 / 模板 Chip 激活态 |
| Disabled | 融资按钮、扩展导航置灰不可点 |
| Modal | 半透明遮罩 + 毛玻璃模糊 + 弹窗入场 |
| Dropdown | 设置面板展开 / 外点关闭 |
| Skeleton | 顶栏「骨架屏」切换图表骨架 |
| Empty | 顶栏「空数据」切换占位 |
| Risk | 日志与弹窗低饱和红边提醒（非刺眼） |

| 现网 | 本原型 |
