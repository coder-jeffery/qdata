## 2. A1 — 策略实验工厂


| # | 优先级 | 任务 | 状态 | 需求要点 |
|---|--------|------|------|----------|
| A101 | P0 | 实验规格 `ExperimentSpec` | ✅ | factor×weight_method×universe×区间×version 声明式配置（YAML/CLI） |
| A102 | P0 | 批跑编排 `run_factor_matrix` | ✅ | 循环调用既有 `run_backtest`；失败隔离、汇总 exit code |
| A103 | P0 | 结果汇总表 | ✅ | Lake/CH：`experiment_id` + 各 run 关键 metrics 宽表 |
| A104 | P0 | CLI | ✅ | `python -m qdata.jobs.run_experiment_matrix ...` |
| A105 | P1 | Dashboard「实验矩阵」页 | ✅ | 读汇总表；按 factor/method 透视 Sharpe/超额/换手 |
| A106 | P1 | 单测 + 短区间冒烟 | ✅ | 2 因子 × 2 权重 × 现有 07-01～15 可跑通 |
| A107 | P2 | 实验笔记模板 | ✅ | 自动写 `summary.md`（规则模板，对齐研判简报风格） |

**验收：** 一命令产出 ≥4 个 run + 一张对比表；Dashboard 可筛选对比。

**建议默认矩阵（MVP）：**

| 维度 | 取值 |
|------|------|
| factor | `mom_20, vol_20, turn_20, ep, bp`（可选 `mom_60`） |
| weight | `equal`, `industry_neutral` |
| universe | `000905.SH` |
| execution | `next_open` |
| version | 最新已发布 `dataset_version` |
