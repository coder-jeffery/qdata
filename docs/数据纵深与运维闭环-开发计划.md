**建议命令（DD1 编排，推荐）：**

```bash
# 水位 / 缺口
.venv/bin/python -m qdata.jobs.dd1_depth --status

# 下一段未完成季度（默认 continue-on-error + 跳过已发布）
.venv/bin/python -m qdata.jobs.dd1_depth --run-next
# 或指定段：--run 2025Q3 | 2025Q4 | 2026Q1 | 2026Q2 | 2026Q3a

# 全段缺口依次
.venv/bin/python -m qdata.jobs.dd1_depth --run-all

# 整段验收（DD104）
.venv/bin/python -m qdata.jobs.validate_release \
  --start 2025-07-01 --end 2026-07-15
```

等价底层仍可直接用 `prod_backfill --skip-calendar --continue-on-error`。
