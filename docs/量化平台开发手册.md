## 1. 平台定位与架构

**定位：** A 股日频研究 → 纸交易闭环平台。先读数正确、可复现，再谈真金。

```
Fetcher（多源）
  → Raw Lake（Parquet）
  → Loader → ClickHouse
  → DataAPI / 因子 / 回测
  → 应用链路：实验 → 信号 → Paper → 盯市 → 监控
  → Dashboard（Streamlit）
```

**硬约束顺序：**

```
全市场日线稳定发布 → 质检可信 → 选股域/行业 → 种子因子 → 回测读数 → 实时/下单
```

**明确不做（当前）：** 真金实盘 UI、做空/融资、LLM 荐股、分钟/Tick 交易、完整归因平台。

## 3. 模块功能清单

### 3.1 数据层（M1 / M1.5）

| 模块 | 功能 | 入口 / 路径 |
|------|------|-------------|
| Fetcher | akshare/baostock/tushare/joinquant 等；生产禁 auto | `qdata.fetchers` |
| Raw Lake | `data/data-lake/raw/<source>/...` | Parquet |
| Loader | security_master / daily_bar / basic / finance / suspend | `qdata.loaders` |
| 交易日历 | 同步开市日 | `python -m qdata.jobs.sync_calendar` |
| 生产回填 | 全市场日线 + 发布 `dataset_version` | `python -m qdata.jobs.prod_backfill` |
| 发布验收 | 连续性 + smoke | `python -m qdata.jobs.validate_release` |
| 日批 | 盘后无人值守（含 extras / M2 / A405 监控） | `python -m qdata.jobs.daily_run --post-m2 ...` |
| DataAPI | `get_price` / `get_universe` / `get_fundamental` + version | `qdata.api.data_api` |
| extras | basic / finance / suspend 水位增量 | `python -m qdata.jobs.sync_market_extras` |

### 3.2 研究与因子（M2）

| 模块 | 功能 | 入口 |
|------|------|------|
| 种子因子 | mom_20/60, vol_20, turn_20, ep, bp | `python -m qdata.jobs.compute_factors` |
| 指数/行业 PIT | 成分与申万 L1/L2 | `sync_index_member` / `sync_industry_member` |
| 组合权重 | equal / factor_rank / industry_neutral | `qdata.research.portfolio` |
| 个股研判 | 五维打分 + 画像 + 可交易性 + 规则简报 | `python -m qdata.jobs.stock_judgment` |
| 单票技术图 | MA5/10 · K线 · MACD · KDJ · 量 · 布林带 | Web `/chart`；`GET /api/research/ta/{code}`；`qdata.research.ta` |

### 3.3 回测（M2.5）

| 模块 | 功能 | 入口 |
|------|------|------|
| BacktestEngine | T 收盘信号 → T+1 开盘成交（可配） | `python -m qdata.jobs.run_backtest` |
| CostModel | 佣金/印花税/滑点，与 Paper 共享 | `qdata.trading.cost` |
| 归档 | Lake + CH `backtest_run` / equity | `data-lake/backtests/` |

### 3.4 应用链路（M2.7）

| 编号 | 模块 | 功能 | 入口 |
|------|------|------|------|
| A1 | 策略实验工厂 | factor×weight 矩阵回测 + 汇总 | `python -m qdata.jobs.run_experiment_matrix` |
| A2 | 研究信号台 | asof 目标权重落盘 + 暴露/可交易性 | `python -m qdata.jobs.build_signal` |
| A207 | 信号↔研判 | TopN 批量研判 / Dashboard 跳转 | `--signal` / 信号台页 |
| A3 | Paper 调仓 | 权重→整手→Gateway→session | `python -m qdata.jobs.paper_rebalance` |
| A306 | 日终盯市 | 收盘价重估市值/盈亏 | `python -m qdata.jobs.paper_mark` |
| A308 | 实验挂钩 | `--from-experiment` 选优→信号→调仓 | 显式开关，默认关 |
| A4 | 因子监控 | 覆盖率 / 分层 / 告警落盘 | `python -m qdata.jobs.monitor_factors` |
| A405 | 日批软挂监控 | `--post-m2` 后监控；不阻断发布 | `daily_run` |
| A5 | Paper 运营台 | 账户/成交/拒单/盯市 | Dashboard「Paper运营」 |
| A504 | 多 session 对比 | 横向对比资产/成交/实验挂钩 | Dashboard multiselect |

### 3.5 交易预备（M3 纸交易）

| 模块 | 功能 | 入口 |
|------|------|------|
| TradingGateway | 风控前置 + 下单 | `qdata.trading` |
| PaperBroker | 模拟成交 + CostModel | `qdata.brokers.paper` |
| RiskLimits | 整手/涨跌停/停牌/限额 | `qdata.risk` |
| realtime | 分通道快照落 Lake | `python -m qdata.jobs.realtime_snapshot` |
| 真金通道 | miniqmt / easytrader 骨架 | **不上真金**（4.4 ❌） |

### 3.6 Dashboard（M2.6 + 应用页）

启动：`python -m qdata.jobs.dashboard`（需 `pip install -e '.[dashboard]'`）

| 页 | 内容 |
|----|------|
| 数据健康 | 水位 / 发布 / 漂移 |
| 因子覆盖 | 覆盖率 / 分位 / 矩阵 |
| 选股域/行业 | 指数规模 / 行业分布 |
| 财务 PIT | 公告水位 / 科目 |
| 单票研究 | 行情 / 估值 / 因子时序 |
| 个股研判 | 五维 + 简报 |
| 回测报告 | 最近 run 对比 |
| 实验矩阵 | A1 汇总透视 |
| 信号台 | 权重 + A207 研判联动 |
| 因子监控 | 覆盖/告警（含日批来源） |
| Paper运营 | session 明细 + 盯市 + A504 对比 |

## 4. 关键命令速查

```bash
# 日批（含 A405 软挂监控）
.venv/bin/python -m qdata.jobs.daily_run \
  --post-m2 --with-basic --with-extras --sync-industry --industry-levels sw_l1,sw_l2

# 实验矩阵
.venv/bin/python -m qdata.jobs.run_experiment_matrix \
  --start 2026-07-01 --end 2026-07-15 --factors mom_20,ep --weight-methods equal,industry_neutral

# 信号
.venv/bin/python -m qdata.jobs.build_signal --date 2026-07-15 --factor ep --top-n 50

# 信号 TopN 研判（A207）
.venv/bin/python -m qdata.jobs.stock_judgment --signal data/data-lake/signals/2026-07-15/<id>/ --top-n 10

# Paper 调仓
.venv/bin/python -m qdata.jobs.paper_rebalance --date 2026-07-15 --factor ep --cash 1000000

# 实验最优 → Paper（A308）
.venv/bin/python -m qdata.jobs.paper_rebalance --from-experiment exp_xxx --date 2026-07-15

# 日终盯市（A306）
.venv/bin/python -m qdata.jobs.paper_mark --latest

# 因子监控
.venv/bin/python -m qdata.jobs.monitor_factors --date 2026-07-15
```

## 5. 数据契约（Lake / CH）

| 产物 | 路径 / 表 |
|------|-----------|
| 日线发布 | CH `daily_bar` + `dataset_version` |
| 因子 | CH `factor_value` |
| 实验 | `lake/experiments/<exp_id>/`（summary/meta）+ CH `experiment_run` |
| 信号 | `lake/signals/<asof>/<signal_id>/` |
| Paper | `lake/paper_sessions/<session_id>/`（orders/positions/marks/meta） |
| 因子监控 | `lake/factor_monitor/<date>/report.json`（`via=daily_run` 表示日批） |
| 回测 | `lake/backtests/` + CH `backtest_run` |

**审计字段示例：**

- Paper `meta.from_experiment`：experiment_id / selected_cell / rank_by  
- 监控 `report.via`：`daily_run` | `cli`  
- 回测 / 信号 meta：`dataset_version`、`app_pipeline_version`（`app-0.1`）
